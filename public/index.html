What is MoltPal?

MoltPal gives AI agents the ability to spend money autonomously within controlled budgets.

No more manual approvals. No more payment bottlenecks. Just smart, controlled, autonomous spending.

💰 Budget Controls Set per-transaction, daily, and monthly limits
🔒 Secure API keys hashed, rate-limited, audited
📊 Real-time Instant balance checks and transaction logs
🚨 Emergency Stop Kill switch for rogue agents

How It Works

  • 1. Fund your MoltPal account
  • 2. Create an API key for your agent
  • 3. Set spending limits (daily, monthly, per-tx)
  • 4. Your agent makes payments via API
  • 5. Monitor all transactions in real-time

Quick Start

Get up and running in seconds with Docker:

git clone https://github.com/alallos/moltpal.git cd moltpal docker-compose up -d

API will be available at http://localhost:3000

View on GitHub →

Example Usage

Agent making a payment:

curl -X POST http://localhost:3000/api/payment/pay \ -H "Content-Type: application/json" \ -H "X-API-Key: molt_your_key_here" \ -d '{ "amount": 5000, "description": "OpenAI API usage", "merchant": "OpenAI" }'

Response:

{ "success": true, "transaction": { "id": "uuid", "amount": 5000, "status": "completed" }, "balance": 95000 }
